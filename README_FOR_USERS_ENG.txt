Sofia supports phrases:
Phrases activation: Софія, Софа, Cофійка, Софі.
Phrases deactivation: Закрийся, тихо, допобачення, папа, Софія закрийся, вирубайся, закрився.
Phrases how now time?: яка зараз година, Софія яка зараз година, яка година, Софія яка година.
Phrases open google: Софія відкрий гугл, включи гугл, відкрий гугл, відкрий інтернет.
Phrases open youtube: софія відкрий ютуб, відкрий ютуб, ютуб.
Phrazes create notes: софія зроби замітку, зроби замітку, створи замітку.
Phrases how now weather?: яка погода, яка зараз погода, Cофія яка погода, Cофія яка зараз погода.
Phrases how now temperature: скільки зараз градусів, скільки зараз градусів на дворі, яка зараз температура, яка температура на дворі.
